depth is estimated automatically in your browser (Depth Anything V2, ~25MB one-time model download). bright pixels = near (unless inverted). you can also supply your own grayscale depth map or a .ply splat via the Depth Source button.
ml-sharp: how to produce a splat 
git clone https://github.com/apple/ml-sharp
cd ml-sharp
pip install -e .
sharp predict -i your_image.jpg -o output/
produces output/your_image.ply. upload the .ply via the Depth Source button to load it.
